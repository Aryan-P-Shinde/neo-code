# Redrob Intelligent Candidate Ranker

**Redrob Hackathon — Intelligent Candidate Discovery & Ranking Challenge**

## Core idea

Most candidate ranking systems score what a profile *says*. This system scores what a candidate's history *proves*.

Three-layer scoring architecture:
1. **Role Fit** — title trajectory, career description evidence, experience range, location, company background
2. **Evidence Quality** — cross-references skill claims against duration, assessment scores, endorsements
3. **Availability** — behavioral signals: activity recency, response rate, notice period

The anti-gaming layer: career description text is the hardest thing to fake. A candidate who mentions "embedding drift," "NDCG," "A/B testing ranking models" in their job descriptions worked on these things. One who lists "Pinecone" as an expert skill with 0 months duration and no matching job description probably didn't.

## Run the ranker

```bash
# Install dependencies
pip install -r requirements.txt

# Run on full candidate pool (produces submission.csv)
python rank.py --candidates candidates.jsonl --out submission.csv

# Run the recruiter demo UI
streamlit run app.py
```

## Files

- `ranker.py` — core scoring engine
- `rank.py` — CLI entry point (thin wrapper)
- `app.py` — Streamlit recruiter demo
- `top200_results.json` — pre-computed results for demo (generated by ranker)
- `submission_v2.csv` — submission file

## Reproduce submission

```bash
python rank.py --candidates ./candidates.jsonl --out ./submission.csv
```

Runs in ~90 seconds on CPU. No GPU required. No API calls during ranking.

## Scoring architecture

```
FINAL SCORE = (Role Fit × Evidence Quality Multiplier × Availability Multiplier × 0.92)
            + (Tiebreaker × 0.08)

Role Fit components:
  Title/Trajectory      25%  — current title + career progression toward ML/AI
  Career Evidence       35%  — technical terms in job descriptions (not skills section)
  Experience Fit        15%  — YoE vs JD's 5-9yr range
  Location              10%  — Pune/Noida/preferred India cities + relocation
  Company Background    15%  — product vs consulting career history

Evidence Quality (multiplier 0.6–1.25):
  — Skill proficiency vs duration consistency
  — Assessment scores vs claimed proficiency
  — Core skill depth (12+ months in JD-critical skills)

Availability (multiplier 0.45–1.0):
  — Days since last active (35% weight)
  — Recruiter response rate (30% weight)
  — Notice period (20% weight)
  — Open to work flag (10% weight)
  Hard ceiling: inactive 6mo + response rate <15% → max 0.45
```

## Honeypot defense

The ranker checks for profile integrity issues before scoring:
- Claimed YoE exceeds career span implied by start dates
- Expert-level skills claimed with 0 months duration
- Assessment scores contradict claimed proficiency (3+ mismatches)
- Current job duration impossible given start date

Candidates flagged as honeypots receive score 0.05 and are excluded from the shortlist.
